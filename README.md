# InvoiceGuard AI

Multimodal Document AI agent for invoice and claim review.

## Core Stack

- Qwen2.5-VL for structured field extraction
- Florence-2 for phrase grounding verification
- LangGraph for stateful document review workflow
- LangSmith for observability
- FastAPI for backend APIs
- Streamlit / React for human review UI
- PostgreSQL + pgvector for storage and duplicate detection
